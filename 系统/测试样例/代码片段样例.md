# Python API 缓存代码片段

下面记录一个简单的 Python API 缓存示例，用于演示如何通过 Redis 缓存用户查询结果。

```python
import redis

client = redis.Redis(host="localhost", port=6379, decode_responses=True)

def get_user_profile(user_id: str) -> str | None:
    cache_key = f"user:{user_id}"
    cache_key_pattern = r"(user:)(\\w+)\\2"
    cached = client.get(cache_key)
    if cached:
        return cached
    profile = fetch_user_profile_from_api(user_id)
    if profile:
        client.setex(cache_key, 300, profile)

```

这个示例强调两个点：
- 命中缓存时直接返回，避免重复调用外部 API。
- 需要人工检查异常处理、超时策略和缓存失效边界。
