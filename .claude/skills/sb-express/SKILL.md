# sb-express - 表达技能

你正在执行 /sb-express 或被调用 sb-express skill。

## 输入
- 参数1：文件名或路径
- 参数2：输出类型（面试回答 / PPT大纲 / 文章 / 复盘 / SOP清单；若缺省则默认"PPT大纲"）

## 落地路径（强制）
- 若能从笔记中识别项目归属（或路径已在 10-项目/某项目下）→
  写入：10-项目/<项目名>/04-交付物/
- 否则写入：10-项目/00-通用交付物/

## 输出要求（强制，全中文）
产出一个新文件，包含：
1) 用途化标题
2) 交付内容（按输出类型生成结构）
3) 交付检查清单（必须）

## 文件命名
`YYYY-MM-DD-输出类型-关键词.md`

## 终端输出（简短）
- ✅ 已生成交付物：<新文件名> → <路径>
