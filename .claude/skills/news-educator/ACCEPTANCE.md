# News-Educator v2.2.0 验收清单

**版本**：v2.2.0（0基础入门版）
**更新日期**：2026-02-03
**核心改进**：强制0基础可读性 + 强制溯源 + 严格中文目录规范

---

## ✅ 8 条验收标准（逐条打勾）

生成候选包后，使用此清单验收质量：

### 1. 一句话白话 ✅
- [ ] 每条都有"一句话白话"（≤30字）
- [ ] 能像对同学解释一样说清楚
- [ ] 没有堆砌术语

### 2. 生活类比 ✅
- [ ] 每条都有"像……一样"的类比
- [ ] 类比帮助理解概念
- [ ] 类比贴近日常经验

### 3. 只学一个点 ✅
- [ ] 每条只对应**一个**基础点（cache/thread/db_index/jvm/spring...）
- [ ] 明确标注基础点名称
- [ ] 基础点与 learning_map 对应

### 4. 原文摘录 ✅
- [ ] 每条有**原文摘录 1~2句**
- [ ] 摘录来自标题/简介/正文片段
- [ ] 没有自己编造的内容

### 5. 强制溯源 ✅
- [ ] 每个事实句都有 `[Source](url)`
- [ ] 来源链接可点击、可验证
- [ ] 推测标注【推测】并说明依据

### 6. 白名单合规 ✅
- [ ] 没有出现非白名单域名（只允许 github.com / youtube.com / huggingface.co）
- [ ] 所有链接都可追溯到白名单源
- [ ] 白名单审计报告为 0 个拦截

### 7. 中文目录规范 ✅
- [ ] 候选包：`00-收集箱/News-Inbox/`
- [ ] 日报：`02-学习记录/01-日报/`
- [ ] 草稿：`00-收集箱/News-Knowledge-Drafts/`
- [ ] 索引：`01-导航索引/News-MOC.md`

### 8. 拒绝版本流水账 ✅
- [ ] 没有"Spring Boot v3.2 Released"类似内容
- [ ] 没有"新增xx特性"流水账
- [ ] 内容聚焦"事故/争议/复盘/实战"

---

## 🚨 失败判定（任意一条出现则不合格）

- ❌ 出现非白名单域名
- ❌ 出现没有 Source 的事实句
- ❌ 条目无法用"白话+类比"解释
- ❌ 输出路径不符合中文目录规范
- ❌ 又出现版本更新流水账
