# sb-capture - 捕获技能

你正在执行 /sb-capture 或被调用 sb-capture skill。

## 目标
把输入内容写入 Obsidian 收集箱：`00-收集箱/`，并严格使用模板 `80-模板/Capture-模板.md`。

## 输入
用户提供的内容（可能是文本、链接、摘录、灵感）。

## 步骤（强制）
1. 读取用户输入
2. 生成文件名：`YYYY-MM-DD-关键词.md`（关键词从输入中提取 1~4 个中文词，避免太长）
3. 读取模板 `80-模板/Capture-模板.md`，用以下规则填充：
   - {{date}}：当天日期（YYYY-MM-DD）
   - {{url}}：若输入包含链接则填链接，否则写"未提供"
   - {{title}}：一句用途化标题（不要纯知识点）
   - {{content}}：原始输入原样放入
4. 在 `00-收集箱/` 创建该文件并写入内容
5. 输出必须简短，仅输出一行：
   - ✅ 已捕获：<文件名> → 00-收集箱/

## 严谨性
- 不得编造来源；没有链接写"未提供"
- 不得输出长篇解释
