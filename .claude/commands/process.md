# Process Obsidian Inbox

处理 Obsidian 收件箱中的笔记，运行自动化处理流程。

## 使用方法

```
/process [选项]
```

## 选项

- `--scan` 或无参数: 扫描收件箱，查看待处理笔记
- `--process`: 处理下一条笔记
- `--status`: 查看队列状态
- `--dry-run`: 预览模式，不实际执行移动

## 示例

```
/process --scan      # 扫描收件箱
/process --process   # 处理下一条
/process --status    # 查看状态
/process --dry-run   # 预览处理结果
```

## 工作流程

1. **扫描**: 扫描 `00-收集箱/` 目录
2. **安全检查**: 检测敏感信息
3. **生成摘要**: 提取标题和 TL;DR
4. **生成标签**: 基于关键词生成标签
5. **分类**: 基于关键词分类到目标目录
6. **语义路由**: 映射到具体领域子目录
7. **移动**: 移动文件到目标目录并更新 Frontmatter
