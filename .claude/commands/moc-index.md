# MOC Index（索引挂载）

## 输入
- 新笔记标题 + 路径（若没有路径先建议路径）
- 笔记内容摘要（可选）

## 输出
1) 推荐挂载到哪个 MOC（给出 1 个主 MOC + 可选 1 个副 MOC）
2) 生成可粘贴的 MOC 片段（带分组标题）
3) 若该主题 MOC 仍"待补充"，生成一个占位块：
   - 待补充问题
   - 参考来源/章节
   - 下一步行动（最小可执行）

## MOC 路由规则

### 核心MOC位置
- **总仪表盘**：`01-导航索引/00-总仪表盘.md`
- **领域总览**：`01-导航索引/01-领域总览-MOC.md`
- **项目总览**：`01-导航索引/02-项目总览-MOC.md`
- **输出作品总览**：`01-导航索引/03-输出作品总览-MOC.md`
- **Java知识地图**：`10-领域知识/01-计算机/Java/Java-MOC.md`

### 挂载规则
| 笔记类型 | 主MOC | 副MOC（可选） |
|---------|-------|--------------|
| Java基础知识 | Java-MOC | 领域总览-MOC |
| Java面试题 | Java-MOC | 输出作品总览-MOC |
| 项目笔记 | 项目总览-MOC | Java-MOC（如果技术相关） |
| 面试回答 | 输出作品总览-MOC | Java-MOC |
| 学习记录 | 总仪表盘（最近更新） | - |

## MOC片段模板

### Java-MOC 挂载示例
```markdown
### {{子分类}}（如：01-基础、02-并发等）
- [[笔记标题]] - {{一句话描述}}
```

### 输出作品总览-MOC 挂载示例
```markdown
### {{类型}}相关（如：缓存相关、Java基础等）
- [[笔记标题]] - {{一句话描述}}
```

### 待补充占位块模板
```markdown
### {{主题}}（待补充）
- [ ] **核心问题**：{{要回答的问题}}
- [ ] **参考来源**：{{书籍/课程/文档}}
- [ ] **下一步**：{{最小可执行动作}}
```

## 规则
- 不要把同一笔记挂到 3 个以上位置（避免索引膨胀）
- 结构优先：学习路径 > 平铺目录
- 每个挂载点必须有一句话描述（不超过15字）
- 如果MOC不存在，先建议创建MOC
- 优先维护"最近更新"区域（总仪表盘）
