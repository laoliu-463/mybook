# Output Crafter（输出作品生成）

## 输入
- 输出类型：面试回答 / 技术文章 / 课程报告 / PPT 大纲
- 主题
- 参考材料（可贴笔记/知识卡链接或内容）

## 输出（按类型模板）


### 类型3：PPT大纲
```markdown
# {{演讲主题}} - PPT大纲

## 第1部分：开场（1-2分钟）
### Slide 1：标题页
- 标题：{{主标题}}
- 副标题：{{副标题}}
- 演讲人 + 日期

### Slide 2：问题引入
- 核心问题：{{一句话问题}}
- 痛点描述（3个bullet points）

## 第2部分：主体内容（8-10分钟）
### Slide 3：解决方案概览
- 方案框架图（Mermaid/图片）
- 3个核心要点

### Slide 4-6：技术细节
每张聚焦一个点：
- Slide 4：{{要点1}}
- Slide 5：{{要点2}}
- Slide 6：{{要点3}}

### Slide 7：实际案例
- 项目背景
- 应用效果（数据）

## 第3部分：总结（1-2分钟）
### Slide 8：效果总结
- 核心收益（量化）
- 适用场景

### Slide 9：Q&A
- 预期问题清单

## 演讲脚本要点
- 开场hook：{{吸引注意力的开场}}
- 过渡语：{{各部分之间的衔接}}
- 结束语：{{行动号召}}
```

## 规则
- 不确定内容必须标注，不要编造
- 输出要"可复用"：能直接拿去复习/投递/面试
- 面试回答必须结合项目经验
- 技术文章必须有代码示例和数据
- PPT大纲要控制时长（每张slide预估时间）

- 课程报告 → `30-输出作品/03-课程报告与文档/`
