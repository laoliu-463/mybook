# process-paper

适用内容：
- 论文阅读笔记
- 含 Abstract、Method、Conclusion 等结构化研究内容
- 需要提取研究背景、方法、结论、局限性的资料

输出要求：
- 保留原文正文
- 用中文补全 TL;DR
- 推荐 `type: resource` 或 `type: overview`
- 提取 3-8 个高价值标签
- 标记需要人工复核的事实或结论
