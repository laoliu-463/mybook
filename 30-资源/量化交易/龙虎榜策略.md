# 龙虎榜策略

## 核心定义

龙虎榜是沪深交易所每日收盘后公布的异动股票交易汇总信息，旨在披露资金动向，增加市场透明度。仅披露买入和卖出金额最大的前五名席位。^[1,15]

## 关键要点

### 1. 触发条件（满足任一即上榜）

| 指标 | 阈值 | 说明 |
|------|------|------|
| **价格偏离值** | ±7% | 偏离值 = 个股涨跌幅 - 对应分类指数涨跌幅^[1] |
| **换手率** | 20% | 筹码充分交换，主力可能进场/出货^[2] |
| **振幅** | 15% | 多空博弈激烈^[2] |
| **连续异动** | 3日累计±20% | 或10日累计100%^[2,3] |

### 2. 核心数据字段

| 字段 | 含义 |
|------|------|
| **席位名称** | 资金身份（机构专用/营业部/深股通专用） |
| **买入/卖出金额** | 各席位单边成交额 |
| **净买入额** | 买入金额 - 卖出金额 |
| **买卖总额比** | 判断多空力量的核心指标^[6] |

### 3. 席位类型识别

| 席位类型 | 特征 | 信号含义 |
|----------|------|----------|
| **机构专用** | 基金/社保/保险，看重基本面 | 数量越多、金额越大，后续上涨确定性越高^[4] |
| **沪深股通** | 北向资金，"聪明钱" | 机构+北向联合买入是极强共振信号^[10,11] |
| **知名游资** | 如"章盟主"，擅长波段 | 可能持有3-5天^[9,10] |
| **短线游资** | "一日游"风格 | 今买明卖，次日砸盘风险高^[4] |
| **散户营业部** | 不知名营业部扎堆 | 跟风行情，难以持续^[11] |

## 量化策略建模

### 资金均衡度模型

```
健康结构：
- 买一至买五金额占比均衡（各15%-25%）
- 买一占比 < 50%（避免"一家独大"）
→ 代表"合力做多"，接力意愿强

风险结构：
- 买一占比 > 60%，其他席位极小
→ "独食"风险，次日无人接盘易砸盘
```
^[6,7]

### 多空力量比率

**理想多头信号：**
- 买入前五总额 ÷ 卖出前五总额 ≥ **2:1**
- 卖一金额 < 买入总额的 **30%**
- 表明抛压轻、做多坚决^[7]

### 席位协同效应

同一板块多只股票同时出现知名游资或机构席位 → 板块资金共识，可提升策略权重^[9,10]

## 具体交易信号

### 策略一：三连板爆量分歧低吸

| 步骤 | 条件 |
|------|------|
| **筛选** | 3连板及以上（非连续一字板） |
| **确认** | 首次爆量分歧（成交量创新高，盘口有烂板/剧烈震荡） |
| **龙虎榜验证** | 当晚显示新资金进场承接，换手充分 |
| **买点** | 次日竞价或开盘下杀时段（水下-4%至-6%）低吸 |
| **逻辑** | 博弈弱转强^[12] |

### 策略二：机构游资共振

| 信号 | 龙虎榜显示"机构专用"与"知名游资"共同买入 |
|------|------|
| **逻辑** | 游资点火拉升 + 机构锁仓趋势 = 波段行情^[4,8] |

### 策略三：成本线回踩

通过3日龙虎榜数据计算主力平均成本：
```
主力成本 = 净流入额 / 成交量
```
股价回调至成本线附近且无大额流出 → 低吸机会^[10]

## 常见误区与风险

> [!warning] 净买入假象（对倒出货）
> **陷阱**：净买入为正，但买卖总额接近（如买1亿卖9000万）
> **真相**：主力"左手倒右手"制造虚假繁荣，实则掩护出货^[7]

> [!warning] "假机构"席位
> **特征**：买单金额异常整齐（888万、1111万），尾盘突击买入
> **真相**：量化私募马甲，做隔夜T+1套利，次日大概率砸盘^[11]

> [!warning] 盲目追高
> 龙虎榜是盘后公开数据，次日主力利用"看榜跟风"心理高开诱多
> **切忌**：次日大幅高开时追涨，容易接盘被套^[16]

> [!warning] 忽视板块效应
> 即便个股数据豪华，若板块整体走弱，大概率是"孤狼"行情^[10]

## 面试考点

**Q: 龙虎榜策略的核心逻辑是什么？**

> 通过分析异动股票的席位资金结构，识别机构与游资的协同效应，在多空力量失衡（买卖比≥2:1）且资金均衡度健康时入场，规避对倒出货和假机构陷阱。
