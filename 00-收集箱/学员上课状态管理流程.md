## 学员上课状态管理 - 流程图

### 总览

```mermaid
flowchart TB
    center("学员上课状态管理")
    center --> biz1["业务一：获取列表<br/>GET /evaluateLog"]
    center --> biz2["业务二：点名签到<br/>POST /rollCallBatch"]
    center --> biz3["业务三：停课复课<br/>POST /stop | /reopen"]
    center --> biz4["业务四：还原<br/>POST /rollbackCourseNum"]
```

---

### 业务一：获取列表流程

```mermaid
flowchart TB
    START1(["开始"]) --> A1["GET /evaluateLog"]
    A1 --> B1["接收查询参数"]
    B1 --> C1["创建DTO"]
    C1 --> D1{page>0?}
    D1 -->|是| E1["getList分页"]
    D1 -->|否| E2["getAll查询"]
    E1 --> F1["Mapper SQL查询"]
    E2 --> F1
    F1 --> G1["手机号脱敏"]
    G1 --> H1{分页?}
    H1 -->|是| I1["PaginationVO"]
    H1 -->|否| I2["List返回"]
    I1 --> J1(["结束"])
    I2 --> J1
```

---

### 业务二：点名签到流程

```mermaid
flowchart TB
    START2(["开始"]) --> A2["POST /rollCallBatch"]
    A2 --> B2["接收签到列表"]
    B2 --> C2{校验状态枚举}
    C2 -->|非法| D2[返回错误]
    C2 -->|合法| E2["获取教师ID"]
    E2 --> F2["遍历处理记录"]
    F2 --> G2{记录存在?}
    G2 -->|存在| H2["更新签到"]
    G2 -->|不存在| I2["新建记录"]
    H2 --> J2{需要消课?}
    I2 --> J2
    J2 -->|是| K2["扣减课时"]
    J2 -->|否| L2["不扣课时"]
    K2 --> M2["保存记录"]
    L2 --> M2
    M2 --> N2(["结束"])
    D2 --> N2
```

---

### 业务三：停课/复课流程

```mermaid
flowchart TB
    START3(["开始"]) --> A3{操作类型}
    A3 -->|停课| B3["POST /stop"]
    A3 -->|复课| C3["POST /reopen"]
    B3 --> D3["接收课次ID"]
    C3 --> D3
    D3 --> E3["批量查询课次"]
    E3 --> F3["遍历课次"]
    F3 --> G3["设置状态"]
    G3 --> H3["发送通知"]
    H3 --> I3{继续?}
    I3 -->|是| F3
    I3 -->|否| J3["批量更新"]
    J3 --> K3(["结束"])
```

---

### 业务四：还原流程

```mermaid
flowchart TB
    START4(["开始"]) --> A4["POST /rollbackCourseNum"]
    A4 --> B4["接收ID列表"]
    B4 --> C4["遍历校验ID"]
    C4 --> D4{存在且有效?}
    D4 -->|否| E4[抛异常]
    D4 -->|是| F4["添加到处理列表"]
    E4 --> X4(["失败"])
    F4 --> G4["遍历处理"]
    G4 --> H4["重置状态"]
    H4 --> I4{实扣>0?}
    I4 -->|是| J4["返还课时"]
    I4 -->|否| K4["记录日志"]
    J4 --> K4
    K4 --> L4(["成功"])
```

---

### 四个业务对比

| 业务 | API | 核心操作 | 数据表 |
|------|-----|----------|--------|
| 业务一 | GET /evaluateLog | 条件查询+分页 | lesson_student |
| 业务二 | POST /rollCallBatch | 批量签到+消课 | lesson_student |
| 业务三 | POST /stop/reopen | 状态变更+通知 | lesson |
| 业务四 | POST /rollbackCourseNum | 状态重置+返还 | lesson_student |
