# ML - 模型训练的标准流程是什么

## 一句话结论
机器学习标准流程包括：数据收集与清洗 → 特征工程 → 模型选择 → 训练与评估 → 超参数调优。

## 标准回答
1. **数据收集**：从数据库、API、日志等来源获取数据
2. **数据清洗**：处理缺失值、异常值、重复数据
3. **特征工程**：特征选择、特征变换、特征构建
4. **模型选择**：根据问题类型选择合适模型
5. **模型训练**：使用训练数据拟合模型
6. **模型评估**：用验证集/测试集评估性能
7. **超参数调优**：网格搜索、随机搜索、贝叶斯优化

## 为什么
机器学习不是直接喂数据就能得到好结果，需要系统化的流程确保：
- 数据质量（garbage in, garbage out）
- 特征有效性
- 模型泛化能力

## 流程对比
| 阶段 | 输入 | 输出 | 关键方法 |
|---|---|---|---|
| 数据清洗 | 原始数据 | 干净数据 | 缺失值填充、异常值处理 |
| 特征工程 | 干净数据 | 特征向量 | 归一化、编码、PCA |
| 模型训练 | 特征向量 | 模型 | 监督/无监督学习 |
| 模型评估 | 模型+测试集 | 指标 | 准确率、召回率、F1 |

## 代码示例
```python
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score

# 1. 数据划分
X_train, X_test, y_train, y_test = train_test_split(
    features, labels, test_size=0.2, random_state=42
)

# 2. 模型选择
model = RandomForestClassifier(n_estimators=100)

# 3. 训练
model.fit(X_train, y_train)

# 4. 评估
y_pred = model.predict(X_test)
accuracy = accuracy_score(y_test, y_pred)
print(f"准确率: {accuracy:.2f}")
```

### 代码说明
- `train_test_split` 将数据划分为训练集和测试集
- `RandomForestClassifier` 随机森林分类器
- `fit` 训练模型，`predict` 预测

## 易错点
- 不要用测试集调参，会导致过拟合
- 训练集/验证集/测试集要独立
- 类别不平衡问题要用采样或加权
- 特征工程往往比模型选择更重要
