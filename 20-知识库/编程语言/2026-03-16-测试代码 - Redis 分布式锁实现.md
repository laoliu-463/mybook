## 基本实现

### 1. 加锁

使用 SET 命令配合 NX 和 PX 选项：

```python
import redis
import time
import uuid

class DistributedLock:
    def __init__(self, redis_client, lock_key, timeout=30):
        self.redis = redis_client
        self.lock_key = lock_key
        self.timeout = timeout
        self.token = str(uuid.uuid4())

    def acquire(self, blocking=True, blocking_timeout=10):
        """
        尝试获取锁

        Args:
            blocking: 是否阻塞等待
            blocking_timeout: 阻塞超时时间（秒）

        Returns:
            bool: 是否成功获取锁
        """
        start_time = time.time()

        while True:
            # 使用 SET ... NX ... PX ... 保证原子性
            acquired = self.redis.set(
                self.lock_key,
                self.token,
                nx=True,  # 只有key不存在时设置成功
                px=self.timeout * 1000  # 毫秒为单位
            )

            if acquired:
                return True

            if not blocking:
                return False

            # 检查是否超过阻塞超时
            if time.time() - start_time >= blocking_timeout:
                return False

            # 短暂等待后重试
            time.sleep(0.01)

    def release(self):
        """
        释放锁

        使用 Lua 脚本保证释放锁的原子性：
        只有持有锁的 token 才能释放
        """
        lua_script = """
        if redis.call("get", KEYS[1]) == ARGV[1] then
            return redis.call("del", KEYS[1])
        else
            return 0
        end
        """
        self.redis.eval(lua_script, 1, self.lock_key, self.token)

    def extend(self, additional_time):
        """
        延长锁的过期时间
        """
        lua_script = """
        if redis.call("get", KEYS[1]) == ARGV[1] then
            return redis.call("pexpire", KEYS[1], ARGV[2])
        else
            return 0
        end
        """
        return self.redis.eval(
            lua_script, 1,
            self.lock_key,
            self.token,
            additional_time * 1000
        )
```

### 2. 使用示例

```
# 创建 Redis 连接
redis_client = redis.Redis(host='localhost', port=6379, db=0)

# 创建分布式锁
lock = DistributedLock(redis_client, "my_resource_lock")

# 使用锁
if lock.acquire(blocking=True, blocking_timeout=30):
    try:
        # 临界区代码
        print("获取锁成功，执行任务...")
        time.sleep(5)
        print("任务完成")
    finally:
        lock.release()
else:
    print("获取锁失败")
```

## 最佳实践

1. **设置合理的过期时间**：避免锁永久不释放
2. **使用唯一 token**：确保只有锁持有者能释放
3. **考虑重试机制**：网络抖动时的容错
4. **监控锁等待时间**：及时发现系统瓶颈
