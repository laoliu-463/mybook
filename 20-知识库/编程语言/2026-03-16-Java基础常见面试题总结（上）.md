## 1. 基础概念与常识

### Java语言特点

- 面向对象
- 一次编译，到处运行
- 自动内存管理
- 垃圾回收机制

### JVM vs JDK vs JRE

- **JVM** (Java Virtual Machine): Java 虚拟机，负责运行字节码
- **JRE** (Java Runtime Environment): 运行时环境，包含 JVM + 核心类库
- **JDK** (Java Development Kit): 开发工具，包含 JRE + 开发工具

### 字节码

Java 编译后生成的 .class 文件就是字节码。

### 编译与解释并存

- 编译：源代码 → 字节码
- 解释：字节码 → 机器码

### AOT vs JIT

- **AOT** (Ahead-of-Time): 提前编译
- **JIT** (Just-in-Time): 即时编译

## 2. 基本语法

### 注释形式

```
// 单行注释
/* 多行注释 */
/** 文档注释 */
```

### 标识符 vs 关键字

- 标识符：变量名、类名、方法名
- 关键字：保留字

### Java 关键字

```
abstract, assert, boolean, break, byte, case, catch, char, class, const, continue, default, do, double, else, enum, extends, final, finally, float, for, goto, if, implements, import, instanceof, int, interface, long, native, new, package, private, protected, public, return, short, static, strictfp, super, switch, synchronized, this, throw, throws, transient, try, void, volatile, while
```

### 自增自减运算符

```
int i = 1;
int a = i++;  // 先赋值再加，a=1, i=2
int b = ++i;  // 先加再赋值，b=3, i=3
```

### 移位运算符

```
// 左移 <<
1 << 2  // 4

// 右移 >>
4 >> 1  // 2

// 无符号右移 >>>
-1 >>> 1  // 正数
```

## 3. 基本数据类型

### 8种基本类型

| 类型 | 大小 | 默认值 |
|------|------|----------|
| byte | 1字节 | 0 |
| short | 2字节 | 0 |
| int | 4字节 | 0 |
| long | 8字节 | 0L |
| float | 4字节 | 0.0f |
| double | 8字节 | 0.0 |
| char | 2字节 | '\u0000' |
| boolean | 1位 | false |

### 基本类型 vs 包装类型

- 基本类型：更高效
- 包装类型：支持 null，适合泛型

### 包装类型缓存机制

```
Integer a = 127;
Integer b = 127;
// 缓存范围：-128 到 127
System.out.println(a == b);  // true
```

### 自动装箱与拆箱

```
// 装箱
Integer a = 10;  // 自动装箱

// 拆箱
int b = a;     // 自动拆箱
```

### 浮点数精度丢失

```
0.1 + 0.2 != 0.3  // 精度丢失
```

### BigDecimal 解决方案

```
BigDecimal a = new BigDecimal("0.1");
BigDecimal b = new BigDecimal("0.2");
BigDecimal c = a.add(b);
```

## 4. 变量

### 成员变量 vs 局部变量

| 区别 | 成员变量 | 局部变量 |
|------|----------|----------|
| 位置 | 类中 | 方法中 |
| 默认值 | 有 | 无 |
| 作用域 | 整个类 | 方法内 |
| 内存 | 堆 | 栈 |

### 静态变量

用 `static` 修饰，属于类，所有对象共享。

## 5. 方法

### 静态方法 vs 实例方法

```java
class Demo {
    static void staticMethod() { }  // 静态方法
    void instanceMethod() { }       // 实例方法
}
```

### 重载 vs 重写

- **重载** (Overload): 方法名相同，参数不同
- **重写** (Override): 子类重写父类方法

### 可变长参数

```
void method(String... args) { }
```
