## 什么是静态方法

静态方法是属于类的方法，在类加载时就分配内存，可通过类名直接访问。

## 为什么静态方法不能调用非静态成员

### 核心原因

1. **内存时机不同**
   - 静态方法属于类，在类加载时就分配内存
   - 非静态成员属于实例对象，只有在对象实例化后才存在

2. **调用顺序问题**
   - 当静态方法存在时，非静态成员尚不存在
   - 此时调用内存中还不存在的非静态成员属于非法操作

### 代码示例

```java
public class Example {
    // 静态成员
    public static final char LETTER_A = 'A';
    public static final String GREETING_MESSAGE = "Hello, world!";

    // 非静态成员（实例变量）
    private String instanceVariable = "实例变量";

    // 静态方法
    public static void staticMethod() {
        // ✅ 可以访问静态成员
        System.out.println(LETTER_A);

        // ❌ 不能访问非静态成员
        // System.out.println(instanceVariable); // 编译错误
    }

    // 实例方法
    public void instanceMethod() {
        // ✅ 可以访问所有成员
        System.out.println(LETTER_A);
        System.out.println(instanceVariable);
    }
}
```

## 总结

- 静态方法在访问本类成员时，**只允许访问静态成员**，不允许访问实例成员
- 实例方法**无此限制**
- 调用静态方法建议使用 `类名.方法名` 的方式，无需创建对象即可访问
