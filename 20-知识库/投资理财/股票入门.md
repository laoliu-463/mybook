# 股票 - 基础概念是什么

## 一句话结论
股票是公司所有权的凭证，买股票即成为股东。股票市场是上市公司股权交易的场所。

## 标准回答
- **股票**：公司所有权凭证，买股票 = 成为股东
- **证券交易所**：上交所、深交所、北交所
- **A股**：人民币计价交易的中国股市
- **涨跌停**：单日涨跌幅限制（A股 10%）

## 为什么
股票是资产配置的重要工具：
- 公司成长带来的股价上涨收益
- 股息分红
- 流动性好，随时买卖

## 核心概念
| 概念 | 说明 |
|---|---|
| 股票 | 公司所有权凭证 |
| 股东 | 股票持有者，享有投票权和分红权 |
| 涨停/跌停 | 单日最大涨跌幅（A股 10%） |
| 市值 | 股价 × 总股本 |
| PE（市盈率） | 股价 / 每股收益 |

## 股票分类
| 分类 | 说明 |
|---|---|
| A股 | 人民币计价，上交所/深交所 |
| B股 | 外汇计价 |
| 港股 | 港币计价，香港联交所 |
| 美股 | 美元计价，NYSE/NASDAQ |

## 交易规则
```python
# A股交易基本规则
- 交易时间：9:30-11:30, 13:00-15:00
- 最小交易单位：100 股（1手）
- 涨跌幅：10%（ST股 5%）
- T+1：当天买入次日才能卖出
```

## 易错点
- 股票不代表债务关系（债券才是）
- 股价短期波动大，长期看价值
- 杠杆放大收益也放大风险
- 新手不建议玩杠杆
