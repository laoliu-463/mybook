# 投资新手入门笔记

> 本笔记整合了通俗解释与专业补充，帮助投资新手建立系统认知。

---

## 一、先建立一个最重要的概念：你把钱放到哪里了？

投资世界里，钱的去向大致有几种：

| 形式 | 本质 | 风险特征 |
|------|------|----------|
| 存银行 | 你把钱借给银行，银行给你利息 | 保本保息属性强 |
| 买债券 | 你把钱借给政府或公司，到期收回本金和利息 | 中低风险 |
| 买股票 | 你买下一家公司的部分所有权 | 高波动高风险 |
| 买基金 | 交给基金经理或按指数规则配置 | 取决于底层资产 |
| 买货币基金 | 买一篮子短期货币市场工具 | 低风险 |

**投资的第一步不是"哪里收益高"，而是"我买的到底是什么资产"。**

---

## 二、现金管理类：余额宝、零钱通、货币基金

### 余额宝、零钱通是什么？

**通俗解释**：这类产品绝大部分本质上是货币基金。你表面上看是在"放零钱"，实际上是把钱买进了一只或几只货币市场基金。

**专业补充**：货币基金主要投资于：
- 银行存款
- 同业存单
- 短期债券
- 回购协议
- 央行票据等短期低风险资产

特点：
- 流动性强
- 波动很小
- 收益通常高于活期存款
- **但不等于保本保息**（不受存款保险制度保护）

---

## 三、债券：本质就是"你把钱借给别人"

### 1. 债券是什么？

**通俗解释**：债券就是一张"借条"——你把钱借给别人，对方约定一段时间后还你本金，并按约定支付利息。

**专业补充**：债券核心要素：
- **发行人**：谁借钱
- **票面利率**：承诺的利息
- **期限**：借多久
- **本金**：借多少钱
- **到期日**：什么时候还钱

### 2. 债券的分类

| 类型 | 说明 | 风险 |
|------|------|------|
| 国债 | 借给国家 | 信用风险很低 |
| 企业债/公司债 | 借给公司 | 风险高于国债 |

### 3. 储蓄国债

面向个人投资者，买了以后主要靠持有到期拿利息。通常有凭证式和电子式两类。

### 4. 柜台国债

可以通过银行柜台或电子渠道买卖的国债，交易属性更强，价格会波动。

### 5. 国债逆回购

**通俗解释**：你把短期资金借出去，拿国债作质押，对方到期还钱并付利息。你不是"买了一个国债"，而是做了一笔**短期借钱交易**。

**专业补充**：
- 一般期限很短（1天、2天、3天、7天等）
- 安全性通常较高
- 收益率会随市场资金面波动

> **新手误区**：国债逆回购不是"长期拿着的国债产品"，而是短期闲钱管理工具。

### 6. 可转债

**通俗解释**：可转债本来是一张债券，但在一定条件下可以转换成股票。兼具债券的"保守属性"和股票的"上涨弹性"。

**专业补充**：投资者可以：
- 持有到期收本息
- 在满足条件时转成股票
- 在市场上买卖赚取价差

---

## 四、股票：买的是公司的一部分

### 1. 股票是什么？

**通俗解释**：股票代表你拥有一家公司的**一小部分所有权**。公司赚钱了，你可能通过两种方式获利：
- **分红**
- **股价上涨后卖出赚差价**

**专业补充**：股票投资收益来自两部分：
1. **资本利得**：低买高卖的差价收益
2. **股息红利**：公司利润分配

> 股票价格会波动很大，**不保证本金安全**。

### 2. 为什么股票会涨跌？

因为市场不断重新判断这家公司值多少钱。影响因素包括：公司业绩、行业前景、市场情绪、利率变化、政策变化等。

### 3. A股、B股、H股

| 类型 | 说明 |
|------|------|
| A股 | 在内地交易所上市、以人民币交易 |
| B股 | 在内地交易所上市、以外币计价交易 |
| H股 | 注册地在内地、上市地在香港 |
| 科创板 | 主要服务科技创新企业，波动通常更大 |

> 科创板开通门槛：通常需要 **50万元资产 + 24个月交易经验**

### 4. 最小交易单位：一手 = 100股

---

## 五、基金：把选资产这件事交给规则或专业人士

### 1. 基金是什么？

基金就是很多人把钱凑在一起，交给专业机构去投资。底层资产可以是：股票、债券、黄金、现金类资产、混合资产等。

### 2. 主动型基金 vs 被动型基金

| 类型 | 说明 | 特点 |
|------|------|------|
| 主动型基金 | 基金经理自己选股票、选债券 | 有机会跑赢市场，但更依赖经理能力，管理费高 |
| 被动型基金（指数基金） | 照着指数去买 | 费用低、风格稳定、信息透明、分散风险 |

> **"被动型基金强于主动型基金"** —— 对多数新手来说，长期定投宽基指数基金往往比盲目挑主动基金更容易执行。

### 3. 场内基金 vs 场外基金

| 类型 | 交易方式 | 特点 |
|------|----------|------|
| 场内基金 | 证券交易所买卖（需要证券账户） | 价格随市场波动，实时成交 |
| 场外基金 | 向基金公司或代销平台申购赎回 | 按净值交易，适合定投 |

常见场外平台：支付宝、银行App、天天基金等。

### 4. 什么是指数？

指数是一个"成绩单"或"温度计"，按照规则把一篮子资产组合起来算出数值，反映市场表现。

常见指数：
- **沪深300**：反映A股大盘蓝筹股表现
- **中证500**：反映中盘股表现

## 六、交易基础术语

### 1. K线

K线是记录价格变化的"蜡烛图"，显示一段时间内的：开盘价、收盘价、最高价、最低价。

> **纠正**：K线是价格图表，不是交易动作（开仓、持仓、平仓是交易动作）

### 2. 开仓、持仓、平仓

- **开仓**：开始买入或建立头寸
- **持仓**：已经持有某个资产
- **平仓**：把持有的仓位卖出或对冲掉，结束这笔交易

### 3. 仓位

仓位就是你有多少钱已经投进去了。

> 例如：10万元资金，买了5万元股票，股票仓位就是50%

### 4. 做多 vs 做空

| 术语 | 解释 |
|------|------|
| 做多 | 看涨，先买入，等价格涨了再卖出赚钱 |
| 做空 | 预期价格会跌，先"借来卖掉"，等价格跌了再低价买回还回去 |

> **新手提醒**：做空和杠杆不适合刚入门的投资者。

### 5. 交易成本

| 成本 | 说明 |
|------|------|
| 佣金 | 支付给券商的手续费 |
| 印花税 | 通常在卖出股票时征收 |
| 过户费 | 与证券登记结算相关 |

## 七、核心金融概念

### 1. 资本利得 vs 分红税

- **资本利得**：低价买入、高价卖出赚到的差价。A股个人投资者通常**不单独征收个人所得税**。
- **分红税**：股票分红涉及税收，持有时间越长，税负通常越低。

### 2. 标的资产

金融产品背后真正对应的那个资产。

例如：股票基金的标的资产是股票，债券基金的标的资产是债券。

### 3. 货币时间价值

今天的100元，比一年后的100元更值钱。因为今天的钱可以拿去投资、产生利息。

### 4. 折现率

把未来的钱折算成今天值多少钱时用的那个"打折标准"。

### 5. 到期收益率（YTM）

如果你现在买入一只债券，并一直持有到到期，你实际大概能获得的综合年化收益率。

### 6. 久期

债券价格对利率变化有多敏感。久期越长，债券价格对利率变化越敏感。

## 八、行为与误区

### 1. 追涨 vs 杀跌

- **追涨**：看到价格涨了就赶紧买，怕错过
- **杀跌**：看到价格跌了就恐慌卖出

> "追涨杀跌"本质是受情绪驱动的交易行为，是新手亏损的重要原因之一。

### 2. 投资 vs 投机

| 术语 | 特点 |
|------|------|
| 投资 | 基于分析，判断风险可控前提下长期有合理回报 |
| 投机 | 更看重短期价格波动，想靠买卖时机赚钱 |

> **投机 = 赌博** 这句话适合提醒新手风险，但更准确地说：**高频冲动交易很容易无限接近赌博**。

## 九、给新手的几个关键常识

### 1. 风险和收益通常成正比

| 资产类型 | 风险收益特征 |
|----------|--------------|
| 货币基金 | 低风险低收益 |
| 债券 | 中低风险中低收益 |
| 股票 | 高波动、高风险、高预期收益 |

### 2. 分散比押单一资产更重要

不要把所有钱押在：一只股票、一个行业、一个主题基金上。

### 3. 流动性很重要

不是所有资产都能随时、按理想价格卖出。

### 4. 不懂不买

如果你说不清这个产品赚什么钱、风险来自哪里、在什么情况下会亏，**就先别买**。

### 5. 新手更该先学"别亏大钱"

投资前期最重要的是：建立认知、控制风险、避免大亏、养成长期纪律。

---

## 十、适合新手的理解顺序

1. **货币基金**：理解现金管理
2. **债券/国债逆回购**：理解"借钱收息"
3. **指数基金**：理解分散投资
4. **股票**：理解公司价值与市场波动
5. **可转债/行业基金/主动基金**：理解更复杂产品
6. **杠杆、做空、衍生品**：最后再碰

---

## 一句话总结版

| 产品 | 核心要点 |
|------|----------|
| 货币基金 | 放短期闲钱的低风险工具 |
| 债券 | 把钱借给政府或公司，收利息 |
| 股票 | 买公司的一部分，赚分红和上涨的钱 |
| 基金 | 买一篮子资产 |
| 指数基金 | 按规则买一篮子资产 |
| 国债逆回购 | 短期把钱借出去拿利息 |
| 可转债 | 可以转成股票的债券 |
| 仓位 | 你投出去的钱占总资金多少 |
| 做多 | 买了等涨 |
| 做空 | 借来先卖，等跌了再买回 |
| 追涨杀跌 | 情绪化投资，容易亏钱 |
