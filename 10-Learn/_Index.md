## 学习顺序建议

### 第一阶段：夯实基础
1. Java SE 核心（集合、多线程、JVM）
2. MySQL 基础（SQL、索引、事务）
3. Vue3 基础（组合式 API、响应式）

### 第二阶段：框架精通
1. Spring 全家桶（IoC、AOP、Boot）
2. MyBatis（原理、动态 SQL）
3. Vue3 进阶（状态管理、路由）

### 第三阶段：进阶提升
1. Redis（数据结构、缓存策略）
2. 并发编程深入
3. 分布式基础
