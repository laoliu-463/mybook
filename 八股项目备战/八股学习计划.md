4 周冲刺路线（每天 3～5 小时，按“学 → 写 → 背 → 讲”）
原则：每天都要写代码 + 每天都要讲 5 分钟（对着自己讲）。面试能否过，很大比例取决于你是否能“讲明白”。
第 1 周：Java 基础 + OOP + 集合（面试高频）
目标：写得出代码、讲得清概念、背得出常见问法。
• Day1：环境 + Maven + IDEA + 基础语法（变量/流程/方法/异常）
• Day2：OOP（封装/继承/多态/抽象类/接口）+ equals/hashCode
• Day3：泛型 + 反射基础（知道怎么用，不死磕）
• Day4：集合 1：ArrayList/LinkedList（底层、复杂度、扩容）
• Day5：集合 2：HashMap（哈希、冲突、树化、为什么线程不安全）
• Day6：IO/NIO 基础 + 序列化 + 常见坑
• Day7：周复盘：把“集合/异常/多态”按面试题风格复述一遍（录音）
Java 基础与集合是 JavaGuide 的必背模块之一。
