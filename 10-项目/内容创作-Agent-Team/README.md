## TL;DR

用 7 个 Agent 分工协作，以 NotebookLM 为内容中台，生产小宇宙播客 + 小红书图文，引流私域变现。

- **赛道**：个人成长认知提升
- **目标人群**：大学生、年轻人（18–25岁）
- **核心痛点**：读了很多书/课，没有实质改变
- **内容闭环**：选题 → 深研 → 播客+图文 → 发布 → 私域 → 变现

## Harness 工作原则

参考 [[长时运行Agent的Harness设计]]：

- 每次 session 只完成 **一个 Agent 任务**
- 完成后立即更新 `content-progress.json`
- A7 质检通过才能将 episode 标记为 `done`
- 新 session 开始前先读进度文件，不靠记忆
