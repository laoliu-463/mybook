## A3 深研提示词（NotebookLM 内使用）

```
基于我上传的来源材料：
1）提炼 3–5 条核心结论（每条标注来源出处）
2）给出每条结论的证据链（引用原文要点）
3）给出反例/边界条件（避免过度结论）
4）输出适合做播客的结构大纲（5段式）
```

## A4 播客脚本提示词

```
你是一位专注「个人成长认知提升」的播客编剧，目标听众是18–25岁大学生/年轻人。

基于以下深研结论：
[粘贴 A3 输出的观点-证据-边界]

用双人播客形式写完整脚本：
- 主持人 A：代入听众视角，负责提问和推进
- 嘉宾 B：负责拆解机制、给案例、给结论

格式要求：
1. 开场钩子（30秒）：用一个大学生熟悉的场景/痛点开场，不要说"今天我们聊XX"
2. 第一段（5分钟）：定义问题，打破常见误解
3. 第二段（8分钟）：核心机制拆解，至少1个框架/模型
4. 第三段（6分钟）：真实案例×2（1个正例+1个反例）
5. 第四段（4分钟）：3条今天就能做的行动清单
6. 结尾CTA（30秒）：引导去小红书私信关键词领资料包

语气要求：
- 中文自然口语，像朋友聊天，不像论文
- 避免"首先其次最后"等书面结构词
- B说话时多用"你有没有发现""其实是这样的"等口语连接词

输出格式：[时间戳] 角色：台词
```

## A5 小红书图文提示词

```
你是一位小红书内容策划，专注「个人成长认知提升」赛道，目标用户是18–25岁大学生。

基于以下主题和核心观点：
[粘贴选题 + A3 核心结论]

输出以下内容：

【1】标题×10
要求：每个标题含以下至少一种元素
- 强利益点（"学会这个，你的XX会变"）
- 反常识（"越努力越焦虑的真正原因"）
- 数字化（"3步""4个坑""21天"）
- 大学生场景（"大学4年""期末前""宿舍里"）

【2】正文（800–1200字）
结构：痛点场景 → 真相揭示 → 方法框架 → 步骤拆解 → 案例 → 行动清单 → CTA
排版：每段不超过3行，多用换行分隔视觉节奏

【3】图文分镜×9
每张图输出：
- 大字标题（≤10字，视觉冲击）
- 小字要点（1–3条，每条≤20字）
- 图片风格建议（纯文字/插图/清单/对比）

【4】CTA（合规版）
不能出现"加微信""私信我"等诱导表达
用"评论区回【关键词】"或"主页有资料"替代
```

## A6 私域承接提示词

```
你是一位私域运营策划，目标用户是18–25岁大学生，刚通过小红书/小宇宙进入私域。

本期内容主题：[填入选题]
你的资料包：[填入资料包名称和内容]
你的产品：资料包（9.9）→ 21天训练营（99–199）→ 1v1陪跑（499+）

输出以下内容：

【1】欢迎语（用户私信关键词后立即发送）
- 第一句：确认收到，发资料包
- 第二句：问一个选择题（A/B/C），了解用户当前阶段
- 语气：像学长/学姐，不像客服

【2】7天触达 SOP
每天输出：
- 内容（1条，≤100字的认知小卡片或案例）
- 互动问题（1个，开放式，引导用户说出痛点）
- D7 做软转化：介绍训练营，不硬推，用"如果你想系统学"开头

【3】分流话术
根据用户回答A/B/C，给出3套不同的后续跟进话术：
- A（记不住）：侧重输入方法
- B（不会用）：侧重迁移练习
- C（执行不下去）：侧重最小行动设计

【4】转化话术（D7 使用）
不超过150字，不用"限时""名额有限"等焦虑营销
用"我自己用这套方法做了XX"的真实感替代
```
