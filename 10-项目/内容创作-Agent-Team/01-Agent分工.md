## Agent Team 分工

| Agent | 职责 | 输出 |
|-------|------|------|
| A1 选题策划 | 每周选题池 10–20 个 | 选题 + 爆点角度 + 痛点 + 承诺结果 |
| A2 资料采集 | 每个选题 5 个可验证来源 | 资料包 + 关键摘录 + 引用来源 |
| A3 NotebookLM 深研 | 基于来源生成结构化结论 | 观点-证据-边界 一页纸 |
| A4 播客脚本 | 双人对话脚本 20–30 分钟 | 开场钩子 + 段落节奏 + CTA |
| A5 小红书改写 | 同题改写图文 | 标题×10 + 正文 + 分镜 |
| A6 私域转化 | 承接链路与话术 | 欢迎语 + 7天SOP + 转化路径 |
| A7 质检合规 | 风险检查 | 夸大承诺/敏感词/引流合规 |

## NotebookLM 笔记本规划

- **研究库（长期）**：书、论文、经典文章、个人沉淀
- **选题工坊（每周）**：当周选题资料 + 生成稿件
