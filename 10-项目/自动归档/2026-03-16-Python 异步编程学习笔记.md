## 什么是异步编程

异步编程是一种编程范式，允许程序在等待某些操作完成时继续执行其他任务。在 Python 中，主要通过 `asyncio` 模块来实现。

## 核心概念

### 1. async/await

```python
import asyncio

async def fetch_data():
    print("开始获取数据")
    await asyncio.sleep(2)  # 模拟 I/O 操作
    print("数据获取完成")
    return {"data": "example"}

async def main():
    result = await fetch_data()
    print(result)

asyncio.run(main())
```

### 2. Task 和 Future

Task 是 Future 的子类，用于调度协程的执行。
